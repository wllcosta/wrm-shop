
# Venda 3D

Sistema interno pra controlar custo, precificação, estoque de produtos e vendas de peças impressas em 3D. Não é loja virtual — a venda em si acontece no Mercado Livre, Shopee, TikTok Shop etc. Isso aqui é só o controle por trás.

Backend em Node/Express, banco Postgres no Neon, front em React. Tudo hospedado junto na Vercel.
